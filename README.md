(English)
Topic 706: Developing a Seat Reservation Application for ITFAirway Airline**  

Objectives:
- Use parallel arrays to store relational data.  
- Utilize file handling functions.  
- Implement string processing functions.  

Requirements:
- Write a program in C/C++ to solve the problem.  
- Implement functions to build the required features.  

General Description: 
- ITF-Airway operates an airplane with a maximum number of seats (**MaxSeats**).  
- The airline needs to develop a program to determine available seats for customer reservations on each flight.  
- The seating chart of the airplane is described in a file.
